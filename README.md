# AI-Powered Customer Support Platform

An advanced customer support solution built with cutting-edge AI, voice, and automation features to streamline conversations and enhance user experience.

## Key Features

- **Real-time AI Chat** using Convex Agents (done)
- **Human Handoff & Auto-Close** powered by AI Tools (done)
- **Smart Knowledge Base** with Embeddings + RAG (done)
- **Voice Support** via [VAPI](https://vapi.ai) (done)
- **Secure API Key Storage** using AWS Secrets Manager (done)
- **Team Management** powered by Clerk
- **Authentication** handled by Clerk (done)
- **Subscription Billing** integrated with Clerk
- **Embeddable Widget** for easy integration
- **Operator Dashboard** for managing conversations (done)
- **Developer Toolkit** for embed script customization
- **Multi-Model AI Support**: OpenAI, Anthropic, Grok
- Built with **Next.js 15** (done)
- Powered by **React 19** (done)
- Styled with **Tailwind v4** (done)
- UI Components from **shadcn/ui** (done)
- Monorepo managed with **Turborepo** (done)
- **Error Tracking** with Sentry (done)

## Tech Stack

- **Frontend**: Next.js 15, React 19, Tailwind v4, shadcn/ui
- **Backend**: Convex, AWS Secrets Manager
- **Auth & Billing**: Clerk
- **Error Tracking**: Sentry
- **Monorepo**: Turborepo

## Dashboard

Operators can:
